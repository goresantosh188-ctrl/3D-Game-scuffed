# MY 3D GAME
This is a game I created using Typescript, Vite, React and Three.js (specifically React Three Fiber)
## Controls
Movement - WASD \
Jump - Space \
Build mode toggle - B \
Place Block - RMB (Only when build mode is toggled) \
Destroy Block - LMB (Only when build mode is toggled) \
Place Portal - P \
Destroy All Portals - O
## Features
* Chunk generation - Move out of the map and generate a new chunk with trees!
* Fast Travel - Place portals to move from spot to spot in the blink of an eye!
* Build Mode - Build structures, text, and parkour courses!
## Note
When you first load the website, you cannot move the camera immediately. You have to click the screen, and only when your mouse is no longer visible, can you move your camera around ingame. This is through Pointer Lock Controls. At any point may you wish to unlock your mouse, press Escape.
